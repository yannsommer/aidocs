XPULINK.AI相信好的软件是迭代出来的。AiStudio将广泛听取用户反馈，坚持快速迭代。  

!!! Abstract ""
    **AiStudio 版本号命名规则为：** v 大版本.功能版本. Bug 修复版本，示例如下：

    - v1.0.1 是 v1.0.0 之后的 Bug 修复版本
    - v1.1.0 是 v1.0.0 之后的功能版本
